# 미디어 통신 계층

1. Network Interface and Hardware

* 링크 간 통신
* 이더넷

2. Internetwork

* 네트워크 간 통신
* 라우터를 사용함
    * 라우터 간 통신은 링크를 넘어선 단계이기 때문에 IP가 필요
* IP, ICMP, IGMP, ARP, RARP

3. Transport

* 데이터 통신의 신뢰성(검증)
* TCP, UDP

4. Application

* 데이터의 사용
* FTP, HTTP
